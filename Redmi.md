# PhishAware — Phishing Awareness Training Simulator

> ⚠️ **FOR AUTHORIZED SECURITY TESTING AND EDUCATION ONLY**
> You must have explicit written permission from the system owner before deploying.

## Overview

PhishAware is a lightweight phishing awareness tool that simulates an Instagram login page. It's designed for security professionals to conduct authorized red team exercises and security awareness training.

### Features

- Realistic Instagram login page clone
- Two-Factor Authentication (2FA) simulation
- Admin panel with captured credential logging
- JSON export of all captured data
- Configurable via environment variables
